ESP32入門講座第3回:MH-ET LIVE MiniKit for ESP32でPWM制御を使ってみよう(Windows版) 

# はじめに
第二回のESP32入門講座で告知した通り、パソコンとマイコン間でシリアル通信を行ってみます。この通信形式を活用することで、取得したセンサーの値やプログラムの変数の値などをパソコンなどに表示させることが可能になり段違いにデバックが行いやすくなります。

# シリアル通信の概要
シリアル通信(serial communication)を直訳すると「ひと続きの通信」になりますが、これはこの通信方式の特徴を端的に表しています。相手側にデータを送る際に情報をひと続きつまり順番に送る、Nokolatと送りたければ順番に一本の信号線を使い一文字づつ(正確には1ビットづつ)'N' 'o' 'k' 'o' 'l' 'a' 't'といった具合に送信します。シリアル通信は通信規格の総称で携帯電話の通信規格における3G、4G、5Gのように複数の通信方式があります。シリアル通信はひとつの通信規格を指す言葉ではなく、こうした方式に基づく通信規格の総称であり、携帯電話における3G・4G・5Gのように複数の通信規格が存在します。主にセンサーとマイコン間の通信で使われる12Cやパソコンとマイコン間の通信で使われるUARTなどがあります。今回の講座ではパソコンと通信をしますので、シリアル通信の中でもUART方式の通信を使用します。
