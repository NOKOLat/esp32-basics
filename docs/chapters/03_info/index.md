# はじめに

前回の予告通り、今回のESP32基本講座ではサーボモータの制御方法について学んで行きましょう。サーボモータとは位置や角度を正確にコントロール出来る便利なモータです。サーボモータのような可動部品を扱えるようになると電子工作の自由度が一気に広がります。例えば、nokolatの飛行機やマルチコプター(ドローン)に搭載されている自動投下装置や、CANSATで使用するパラシュート切り離し機構などを作れるようになります。

## サーボモータ制御回路の確認と準備
![サーボモータとマイコンの回路図](ESP32_and_servo.png)

サーボモータを制御するためにはGND、電源(5V~3.2V)、信号線の三本線をマイコンと接続する必要が有ります。一般的に黄色線が信号線(GPIO)、赤色線が(5V)、茶色又黒色が(GND)に対応しています。三本線を指し間違えるとサーボモータやマイコンが壊れてしまう可能があるので、回路の配線後に必ず**配線のダブルチェック**を行いましょう。今回は上図を回路図を参考にしながら、マイコンとサーボモータを接続して回路を完成させて下さい。

**ワンポイントアドバイス**:上図では見やすさのためワイヤーを経由してマイコンと接続していますが、実際はマイコンピンとコネクタは直接接続して大丈夫です。

## サーボモータを制御の仕組み
先ほど、サーボモータは任意の位置や角度を正確にコントロールできるとお伝えしましたがその仕組みはご存知でしょうか。黄色の信号線にPWM信号を流すことによりサーボモータの位置や角度を指定しています。PWMはPulth Width Modulation(パルス幅変調)の略です。PWM制御は電源のオンとオフを高速で切り替えることで電圧を制御する技術です。

![](PWM_Signal.webp)

大まかなイメージとして、PWM信号は上記のグラフのようなパルス信号のことを指しています。三つのグラフを見比べると、全てオンとオフのタイミングが絶妙に異なっていることが読み取れます。
![ESP32マイコンとサーボモータを接続した回路図]()
